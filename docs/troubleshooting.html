Troubleshooting

Common issues and fast fixes for the SB-HPC cluster and EM workflows.

Access and login

  • Cannot log in: confirm your university account is active and your public SSH key is added at https://myrz.uni-osnabrueck.de/.
  • Permission denied: double-check the correct username and that your SSH key is loaded locally.

GUI and web interfaces

  • GUI apps do not open: ensure XQuartz is running on Mac, then connect with ssh -X username@cluster-address.
  • cryoSPARC web UI not loading: set up port forwarding and open the local URL.
ssh -L 39000:localhost:39000 username@cluster-address
http://localhost:39000

Slurm and job issues

  • Job stuck in queue: check partition and resource request with squeue and sinfo.
  • Job ends immediately: review your job output and error logs for the first failure line.
squeue
sinfo

Software loading

  • Command not found after loading: confirm the package exists via spack find.
  • Not sure what is available: list cryo-EM packages with spack list -r cryo_em.
spack find
spack list -r cryo_em

Storage and cleanup

  • Storage full or project too large: clear intermediate results and delete unproductive jobs.
  • Relion trash still full: empty the Trash folder to reclaim space.

See Data Cleanup for detailed guidance.

Data transfer problems

  • Session not appearing: verify the session naming format is correct and matches your project.
  • Links missing in project folder: confirm the project was initialized with init_project.

See Session Guidelines and Data Transfer for details.

Need help?

If you are still stuck, note the exact error message and the command you ran. Share that with the team for faster support.
