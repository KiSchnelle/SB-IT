Data Transfer

Negative-stain Data

For negative-stain data, the transfer is semi-automated(moeller-group only). To transfer the data, you need to run the transfer script on the microscope computer. The script will transfer the data to the cluster and where it will be automatically converted to mrc and clipped to 3700x5000 pixels. The rawdata (tiff and mrc) will be stored under: 

/sbdata/rawdata/JEM/$session_name

In your project folder a session folder will be created under: 

/sbdata/projects/$username/$project/rawdata/negative-stain/$session_name

This folder will include links to the mrc files in the rawdata folder. The session name is the same as the one used for the data collection.

Cryo-EM Data

For cryo-EM data, the transfer is fully automated. The rawdata including Atlas, Gridsquare and gain reference will be stored under: 

/sbdata/rawdata/Elenor/$session_name

In your project folder a session folder will be created under: 

/sbdata/projects/$username/$project/rawdata/cryo/$session_name

This folder will include links to the eer files in the rawdata folder. Also a mrc folder will be created where you can find the Atlas, Gridsquare images. The gain reference will be stored in the gain folder. The session name is the same as the one used for the data collection.
