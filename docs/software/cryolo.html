Using cryolo

Load the cryolo environment

To use cryolo on the SB-HPC cluster, you need to first load the miniforge3 module. You can do this by running the following command in your terminal:

spack load miniforge3

After loading the miniforge3 module, you have to source the conda activation script. To find the path to the conda activation script, you can use the following command:

which conda

This will give you the path to the conda executable. You now have to remove the condabin/conda part from the path and add instead bin/activate to the end of the path. For example, if the output of the which conda command is /path/to/miniforge3/condabin/conda, you can source the activation script like this: 

source /path/to/miniforge3/bin/activate

After sourcing the activation script, you can activate the cryolo environment by running:

conda activate cryolo

Now you can run cryolo commands directly in your terminal.

Napari boxmanager does not support X11 forwarding and has to be installed locally on your PC for more information look at https://cryolo.readthedocs.io/en/stable/installation.html
