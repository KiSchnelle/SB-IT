Using Topaz

Load the topaz module

To use topaz on the SB-HPC cluster, you need to load the Topaz module. You can do this by running the following command in your terminal:

spack load topaz

Then you can run Topaz commands directly in your terminal.

For using topaz with CryoSparc, you have to use the topaz wrapper script under:

/appl/scripts/topaz_cryosparc.sh

You can then use this path in your CryoSparc job submission or configure it project-wide in CryoSparc.

You can just copy this path and paste it into "Path to Topaz executable" in the CryoSparc settings.

Unless you know what your doing please use default configuration for CPUs/GPUs.
