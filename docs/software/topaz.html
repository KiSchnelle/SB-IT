Using Topaz

Load the topaz module

To use topaz on the SB-HPC cluster, you need to load the Topaz module. You can do this by running the following command in your terminal:

spack load topaz

Then you can run Topaz commands directly in your terminal.

For using topaz with CryoSparc, you have to find the executable in the topaz module directory. You can do this by running the following command:

which topaz

This will give you the path to the topaz executable. You can then use this path in your CryoSparc job scripts or configure it project-wide in CryoSparc.

For example, if the output of the which topaz command is /path/to/topaz, you can just copy this path and paste it into "Path to Topaz executable" in the CryoSparc settings.
